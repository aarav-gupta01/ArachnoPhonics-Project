# ArachnoPhonics

## Description
ArachnoPhonics is a hangman-inspired game that prompts the user to guess letters with the goal of fully guessing the randomly chosen word.

## How to run
To run this program, all you need to do is open main.py and click Run.

## Features
- Displays correct guesses on screen with underscores for letters yet to be uncovered
- Advances Spider Animation as wrong guesses pile up
- Allows user to pick difficulty of game at the start
